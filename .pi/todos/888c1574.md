**File:** `model_orchestrator/infrastructure/phala/_cluster.py`, line 522

`json.loads(stdout[json_start:])` fails with `JSONDecodeError` if the CLI appends any text after the JSON object (e.g., `{"app_id":"x"}\nDone!`). The error is caught and results in a provisioning failure, but using `raw_decode` would correctly parse only the JSON portion.

**Fix:**
```python
deploy_result, _ = json.JSONDecoder().raw_decode(stdout[json_start:])
```
